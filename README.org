** Quickstart

If you don't have pipenv installed: =pip install pipenv=.

#+BEGIN_SRC bash
git clone git@github.com:barischrooneyj/msc-tno-uva
cd msc-tno-uva
pipenv install

cd code
python normal_load.py
python influence_line.py
python normal_response.py
python models.py
#+END_SRC
